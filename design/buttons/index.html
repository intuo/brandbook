Buttons

Although there are more colors available to use, it doesn't make sense to have a wide spectrum of colors for more or less the same type of actions. The color of a button must match the context in which it is used, as illustrated in the example below.

Change badge Publish Delete account Upload image 
  <a href="#" class="btn btn-default">Change badge</a>
  <a href="#" class="btn btn-success">Publish</a>
  <a href="#" class="btn btn-danger">Delete account</a>
  <a href="#" class="btn btn-primary">Upload image</a>

Dropdown buttons

Dropdown buttons should provide a meaningful action. The expected behavior is that the action of the button changes to the selected option.

Attending
  • Attending
  • Not going 
  <div class="btn-group">
    <a class="btn btn-default" href="#" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
      <span class="icon ion-android-done"></span> Attending
      <span class="caret"></span>
    </a>
    <ul class="dropdown-menu">
      <li>
        <a href="#" data-toggle="modal">
          <span class="icon ion-android-done"></span> Attending
        </a>
      </li>
      <li>
        <a href="#" data-toggle="modal">
          <span class="icon ion-android-close"></span> Not going
        </a>
      </li>
    </ul>
  </div>

About

Learn more about the Intuo design patterns and guidelines.

What and why

The brandbook is Intuo's internal CSS framework. It includes basic global styling for typography, small components like buttons and tabs, and our general guidelines for writing HTML and CSS.

There’s a lot we can improve upon in the brandbook and feedback is always appreciated. We will occasionally add, remove, or modify things.

Future updates

See the roadmap for a rough outline on what’s slated for future versions the brandbook.

Dependencies

  • Our styles are built with SCSS.
  • Middleman is used to serve the GitHub pages.
  • middleman-syntax for code highlighting.
  • turbolinks for rapid page loading.
  • We use our own Intuo icon font alongside of Ionicons.
  • Intuo's design framework is built on top of Bootstrap Sass 3.
  • Autoprefixer is used to generate vendor-prefixed declarations.

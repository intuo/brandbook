About

Learn more about the Intuo design patterns and guidelines.

What and why

The brandbook is Intuo's internal CSS framework. It includes basic global styling for typography, small components like buttons and tabs, and our general guidelines for writing HTML and CSS.

There’s a lot we can improve upon in the brandbook, and help is always appreciated. While we don’t currently plan on building this out as a competitor to other front-end frameworks, we will occasionally add, remove, or modify things.

Browser support

The brandbook currently supports Internet Explorer 9+ and the latest two versions of Chrome, Safari, and Firefox on OS X and Windows. While not a responsive or mobile-focused project, Mobile Safari and Chrome for Android should render just fine. Support for Linux-based browsers is not strictly guaranteed, but accounted for whenever possible.

Future updates

See the roadmap for a rough outline on what’s slated for future versions the brandbook.

Dependencies

  • Our styles are built with SCSS.
  • Middleman is used to serve the GitHub pages.
  • We use our own Intuo icon font alongside of Ionicons.
  • Intuo's design framework is built on top of Bootstrap 3.
  • Autoprefixer is used to generate vendor-prefixed declarations.
